# Michelle_CCNY_SU2026_Project 1

Duck River Game

Overview

This is a 2D Unity game where the player controls a duck swimming through an endless river. The camera follows the duck while the background moves using a parallax effect.

Objective

Collect as many coins as possible before the timer reaches zero.

* Normal coins increase the score.
* Special coins temporarily increase the duck’s movement speed.
* Whirlpools pull the duck toward their center.
* Reaching the center of a whirlpool removes 10 points from the score.

Controls

* W / Up Arrow: Move up
* S / Down Arrow: Move down
* A / Left Arrow: Move left
* D / Right Arrow: Move right

The duck’s vertical movement is limited so it remains inside the river.

Main Features

* Horizontal and vertical duck movement
* Infinite parallax river background
* Random coin generation
* Special speed-boost coins
* Randomly generated whirlpools
* Whirlpool pulling effect
* Score and penalty system
* Countdown timer
* Game-over screen with the final score

How to Play

Avoid whirlpool centers while collecting coins throughout the river. Try to earn the highest possible score before time runs out.
